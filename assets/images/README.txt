Tempatkan file gambar Anda di folder ini.
Contoh yang dirujuk di kode: assets/images/image.png

Langkah:
1. Salin gambar (PNG/JPG) ke folder ini dan beri nama image.png (atau ubah path di main.dart).
2. Jalankan: flutter pub get (biasanya otomatis setelah edit pubspec).
3. Jalankan aplikasi: flutter run

Catatan: Anda bisa menambahkan beberapa gambar dan cukup cantumkan folder (sudah dilakukan di pubspec.yaml) jadi tidak perlu menuliskan tiap file.
